svrcore
=======

	  <title>Secure PIN handling using NSS crypto</title>
	  <description>svrcore provides applications with several ways to handle secure PIN storage e.g. in an application that must be restarted, but needs the PIN to unlock the private key and other crypto material, without user intervention.  svrcore uses the facilities provided by NSS.</description>
